A work of interactive fiction.

Harry Potter and the Philosopher's stone was created with Inform and has IFID C8A63770-6D0B-11E2-B933-F96839AE4917. To play a work like this one, you need an interpreter program: many are available, among them Zoom for Mac OS X and for Unix; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
